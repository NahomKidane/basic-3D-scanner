Basic 3D Scanner

Basic 3D Scanner is a simple, cross-platform 3D scanning solution using OpenNI compatible devices Kinect for XBOX, Asus XMotion and Prime Sense Carmine. Basic 3D Scanner is developed as a term project for CSE395 lecture in Computer Engineering of Gebze Institute of Technology (http://bilmuh.gyte.edu.tr) and should not be considered as a final 3D Scanner solution.

Compiling

For compiling Basic 3D Scanner, the libraries listed below should be installed to your system.

Required Libraries

  • OpenNI 1.5.40 unstable (for cross platform Kinect for XBOX support)
  • PCL 1.6+

After installing the libraries above, you can generate a makefile in build directory using CMake.

$ cd repo_directory/
$ mkdir build
$ cd build
$ cmake ..
$ make
